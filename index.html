História dos computadores

No Módulo 1 - Primeiros Passos com HTML estudamos uma breve história dos computadores com as principais caracteristicas de cada geração. De acordo com o apresentado a história se divide em 5 gerações e possuem as seguintes caracteristicas.

  1. Primeira Geração - (1951-1959)
    Os computadores desta época funcionavam por meio de circuitos e válvulas eletrônicas. Seu uso era restrito uma vez que eram muito grandes e possuíam auto consumo de energia. Exemplos desta geração são: MARK I, e ENIACC.
    Foi nesta geração que também surgiu o termo Bug que em tradução livre quer dizer inseto. Este termo era utilizado quando algum inseto encostava-se aos circuitos elétricos e causando algum problema durante o processamento dos dados.
  2. Segudna Geração (1959-1965)
    Ainda possuíam grandes dimensões, entretanto começou a usar os transistores, em substituição das válvulas que eram maiores e mais lentas. Neste período começaram a se popularizar seu uso comercial.

  3. Terceira Geração (1965-1970)
    Nesta geração os transistores foram trocados pelos circuitos integrados, o que permitiu a fabricação de computadores menores dimensões.
    Aqui começou a utilização dos chips e a popularização dos computadores pessoais.

  4. Quarta Geração (1971-dias atuais)
    Com a evolução da tecnologia, os computadores diminuíram de tamanho e aumentaram a velocidade de processamento e armazenamento de Dados. Inclusão dos microprocessadores reduzindo o gasto de energia.
    No inicio do milênio com o surgimento dos softwares integrados surgiu os computadores de mãos, ou seja os smartphones, tablets, que incluem a conexão móvel com navegação web.

O que são Clients?

Client no liguagem web

Na Web client é o dispositivo que esta consumindo informação.
O client solicita através do endereço de pagina ao servidor os arquivos HTML, CSS, javascript e imagens. Estes arquivos serão enciados pelo servidor, e ao chegar ao client sera executado pelo navegador. Algumas destas informações são armazenadas em cache, evitando o carregamento total da pagina toda vez que o usuário a solicitar.

Falando Sobre Navegadores

O navegador é instalado do lado cliente, usado para interpretar os arquivos recebidos pelo servidor Web.
Hoje existem diversos navegadores, como exemplo podemos citar Mozilla, Chrome, Internet Explorer Microsoft Edge entre outros.

Aplicações Web

Aplicações web, são soluções criadas e que possuem a internet como meio de comunicação client x server, que usa somente o navegador, sem a necessidade de instalar algum aplicativo.

O que são Servers?

Sobre servidores

Servidrores são equipamentos que fornecem algum tipo de serviço, pode ser ele servidor de e-mail, arquivos/dados, web.
Estes equipamento devem possuir melhores recursos fisicos como memoria ram, disco de armazenamento e processadores, além destes recursos deve conter alguns softwares especiais, como Sistema operacional para servidores e alguns serviços como por exmplo para servidreos web estes devem conter Apache, Nginx.
Atualmente existem empresas que disponibilizam a contratação de servidores para o armazenadas de suas aplicações e dados, sendo este tipo de serviço interessante uma vez que como ja citado antes os equipamentos para servidor terem que ser maquinas exclusivas para este fim, alem de terem que ser bem dimensionadas para o serviços, uma vez que são equipaentos mais caros. Com a contratação de data center, para se realizar upload de seus recursos e softwares é mais rapído do que realizar o upload em sua sede.

Tags font / del e abbr

Neste projeto nos foi sugigerida a utilização das tags <font>, <del> e <abbr>

A tag <abbr> é utilizada para realizar marcação de abreviatura ou acronimo como HTML, CSS.
Usando o atributo title é possivel adicionar uma descrição a abreviação/acrônimo que ira aparecer ao passar o mouse sobre o elemento. Esta tag suporta os atributos Globais do HTML

Codigo
Resultado
<abbr title="Digital Innovation One""> DIO /acronimo </abbr>
DIO

A tag <del> represeta uma parte de texto excluido em um documento. Quando renderizado pelos navegadores usualmente coloca uma linha cortando o texto. Esta tag suporta os atributos Globais do HTML.

Codigo
Resultado
<del> texto a ser apagado </del> texto a ser apagado

A tag <font>, define tamanho cor e tipo de fonte a ser utilizada. Entretanto essa tag foi descontinuada no HTML5, sendo recomendado a utilização das propriedades de fonte do CSS.

Codigo
Resultado
<font size="5" color= "green" face="arial"> Fonte </font> Fonte
